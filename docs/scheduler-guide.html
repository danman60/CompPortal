Studio Code Assignment

Before scheduling begins, assign anonymous studio codes. These codes hide studio names from judges while allowing you to organize routines.

How to Assign Codes:

  1. Open the Assign Studio Codes modal Click the "Assign Studio Codes" button in the scheduler toolbar.
  2. Review the studio list All studios with routines in this competition are displayed with their entry counts.
  3. Auto-generate codes Click "Auto-Assign All" to automatically assign sequential letter codes (A, B, C...).
  4. Or manually assign Type custom codes in each field. Each code must be unique.
  5. Save assignments Click "Save Codes" to apply the assignments.

💡 Why Use Studio Codes?

Anonymous codes ensure fair judging by preventing judges from knowing which studio is performing. Codes appear on schedules, backstage lists, and judge sheets instead of studio names.

⚠ Assign Codes First

Assign studio codes before building your schedule. The codes will appear in the routine pool and scheduled items.

Getting Started

Access the scheduler from your Competition Director dashboard by navigating to the Schedule page.

  1. Select Competition Use the dropdown at the top to select which competition you want to schedule.
  2. Choose a Day Click on the day tabs to switch between competition days. Each day shows its start time and routine count.
  3. Set Start Time Click the pencil icon next to the time to adjust when the day begins.

💡 Pro Tip

Set your day start time before adding routines - all times will automatically recalculate!

The Routine Pool

The left panel shows all unscheduled routines. Use filters and search to find specific routines quickly.

🔍 Search

Type routine names, dancer names, or studio codes to filter

🎯 Filters

Filter by Classification, Age Division, Genre, Entry Size, or Studio

☑ Select All

Bulk select routines matching your current filters

📋 Views

Toggle between Table view (compact) and Cards view (detailed)

Routine Information Displayed:

Column Description
Routine Title and dancer name(s)
Std Studio code (anonymous identifier)
Class Classification (Competitive, Part-Time, Novice)
Size Entry size (Solo, Duet/Trio, Small Group, etc.)
RA Routine age (averaged from dancers)
Age Age division category
Genre Dance style (Jazz, Lyrical, Contemporary, etc.)
Dur Duration in minutes

📝 Notes Icon

Routines with a memo icon have scheduling notes from the studio (e.g., "Please schedule on Day 2"). Hover over the icon to see the full note.

Scheduling Routines

Schedule routines by dragging them from the pool to the schedule area, or use multi-select for bulk operations.

Single Routine:

  1. Click on a routine row This selects the routine and enables dragging.
  2. Drag to the schedule Drop it in the schedule area on the right. It will be added at the end.
  3. Reorder as needed Drag routines within the schedule to change their order.

Multi-Select:

  1. Check multiple routines Use the checkboxes to select multiple routines at once.
  2. Drag the group Drag any selected routine to move all of them together.
  3. Use "Select All" Click "Select All" to select all routines matching your current filters.

💡 Pro Tip

Filter by genre first, then use "Select All" to quickly add all Jazz routines at once!

Adding Blocks

Blocks are non-performance time slots that appear in your schedule. Use them for breaks, awards, and special events.

🏆

Adjudication

Awards ceremonies, judge critiques, announcements

☕

Break

Lunch breaks, intermissions, setup time

🎉

Event

Special performances, guest appearances, transitions

To Add a Block:

  1. Click the block button Choose +Adjudication, +Break, or +Event from the toolbar.
  2. Set the title Enter a descriptive title (e.g., "Mini Awards", "Lunch Break").
  3. Set duration Specify how long the block will take in minutes.
  4. Position it Drag the block to the correct position in the schedule.

Automatic Time Cascade

Times are calculated automatically based on the day start time and each item's duration. You never need to manually enter performance times!

Example Calculation:

Day Start: 08:00
---
#101 Solo Jazz (3 min) .... 08:00
#102 Solo Lyrical (3 min) ... 08:03
☕ Break (15 min) ............. 08:06
#103 Duet/Trio (4 min) ..... 08:21
#104 Small Group (4 min) ... 08:25
---
Day ends at: 08:29

Default Durations by Entry Size:

Entry Size Duration
Solo 3 minutes
Duet/Trio 4 minutes
Small Group / Large Group 4 minutes
Line / Production 5 minutes

💡 Extended Time

Routines with extended time requested will use their custom duration instead of the default.

Saving & Undo

Your changes are saved manually when you click Save. Each save creates a version snapshot you can restore later.

💾 Save Button

Click Save to persist all changes. Appears green when there are unsaved changes.

↩ Undo Button

Restore to the previous version. Each save creates a restore point.

📋 Version History

Access from Actions menu to see all versions with timestamps.

🗑 Reset Day

Clear all routines from the current day (available in Actions menu).

⚠ Don't Forget to Save!

Changes are held in memory until you click Save. Refreshing the page without saving will lose your work.

Publishing Your Schedule

When your schedule is ready, publish it to make it visible to studios and generate shareable links.

  1. Review your schedule Make sure all routines are in the correct order and times look right.
  2. Click the UNPUBLISHED button This opens the publication options.
  3. Choose visibility Published schedules are visible to all registered studios.
  4. Share the link Use the Actions menu to export or share the schedule.

Export Options (Actions Menu):

📄 PDF Export

Generate a printable PDF of the full schedule

📊 CSV Export

Download as spreadsheet for external tools

📧 Email Studios

Send schedule notifications to all studios

🔗 Shareable Link

Copy a public link to the live schedule
